Assignment 3: Digital Elevation Model

Software Used: QGIS, VSCode, Qgis2threejs, Git

Course Context & Objective

The goal of this assignment was to create a map showing an urban area and the extent to which it would be flooded if the water level rose. I created a map of Utrecht, showing which areas would be flooded if the water level rose to 3m NAP.

Technical Skills Learned

  • Creating 3D interactive maps in QGIS using Qgis2thrheejs.
  • Extruding OSM buildings based on DSM data.
  • Uploading large files to github using the terminal.

Results & Analysis

The interactive map below shows a 3D view of the center of Utrecht. The grey blocks are the buildings, along with their approximate height. The map background is an OpenStreetMap tile that has been 'draped' on the digital terrain model in order to show elevation changes like the canals. The blue polygon shows where water would flood the land if it rose to 3m NAP (Amsterdam standard sea level). The opacity of the different layers can be modified using the hamburger menu.

I found this assignment quite easy and straightforward. Elevation models are something I work with quite often and the YouTube tutorial was very clear. In terms of comprehension, the biggest difficulty was determining whether I should export my map to HTML like the tutorial did, or use ArcGIS like I would have done with the old map assignment. I opted for the former. This map was made live by using Qgis2threejs's HTML exporter. Uploading the files to GitHub was the most challenging part of this assignment. I had to install GIT and learn how to use the terminal to upload files from my PC straight to Git's Large File Storage. I had to do this because my scene.js file was over the maximum GitHub upload size, so I had to find an alternative. In the end though, it worked out (or at least I think it did) and I learned something new from it.

I didn't use AI much for this assignment. I only used it to restructure my code while it was still a work in progress, because I was once again getting a bit lost with my indentations and I did not want to fix them all manually. I did not need AI for the mapping part of the assignment, the YouTube tutorial sufficed.

Assignment Questions

1) My region of interest is the Utrecht city center. I chose this area because I live here and I love this city. I was curious to see how vulnerable it would be to flooding, and how much flooding would be needed to significantly affect the city. The tutorial measured flooding in Rotterdam at 1m NAP but since no flooding was visible in Utrecht at that level, I increased to 3m NAP.

2) What was surprising about Utrecht was to see how much water would need to rise to flood the city. I thought that 1m would already significantly affect the city, but I actually had to raise the virtual water level to 3m to start seeing flooded areas. Utrecht is a lot higher than I assumed, and the canal walls protect the city really well! (In theory, assuming they don’t collapse).

3) My final combined file is 110mb. The DTM is 50mb and the DSM is 60mb. The scene.js file I used to make the interactive map is 26mb.

4) My area is 3 square kilometers. It goes from the Dom Tower in the east to the cinema in the west.

5) In this assignment, I learned how to extrude buildings downloaded from OSM, which is fun! I extruded the buildings to turn 2D polygons into 3D models with the height of the DSM file. I also learned how to use Qgis2threejs which I had unsuccessfully tried to use before. This allowed me to model the 3D map and export it as an interactive html + js file. This is what I used to add the interactive map to my website. I did not learn much about downloading elevation data from PDOK and manipulating DTMs and DSMs because I had done that before, but I did learn about the PDOK plugin for QGIS, which is a lot easier than downloading the data from the AHN viewer.

For this assignment, I had originally planned to look at temporary WWII airfields in Normandy to see how much they had compacted the ground. In some cases, there was a visible difference in elevation where the runways and hangars used to be, even 70 years after they were dismantled. However, I ended up doing the flooding assignment instead because I was unable to make a nice interactive map of the elevation data where the airfields had been.
